Dependencies

The actual colord daemon requires:

  • glib 2.14.0
  • polkit 0.9

gnome-color-manager will need all the usual GNOME libs as well. Just make sure you install colord before gnome-color-manager or colord-kde!

Compiling the latest code

You can get the latest colord daemon and GNOME front-end from the public git repositories on github.com: 

git clone git://github.com/hughsie/colord.git
